## 第二步：进入 KiwiVM 控制面板

购买完成后，登录搬瓦工账户后台，路径是：

> **Client Area → Services → My Services**

找到你买的 VPS，点击旁边的 **KiwiVM Control Panel** 按钮，就进了管理面板。

KiwiVM 首页会显示你这台 VPS 的核心信息：
- **Public IP address**：服务器的公网 IP
- **SSH Port**：不是默认的 22，搬瓦工会随机分配一个五位数端口
- 内存 / 硬盘 / 流量使用情况
- 机房位置和运行状态

这两样东西（IP + 端口）要先记下来，后面 SSH 登录要用。

## 第三步：重置 root 密码

搬瓦工购买后**不会**把 root 密码直接发给你。需要自己在 KiwiVM 里手动生成。

操作路径：

> 左侧菜单 → **Root password modification** → 点击 **Generate and set new root password**

⚠️ **关键注意**：
1. 重置密码前，VPS 必须处于**开机状态**
2. 密码只会出现一次，务必立即复制保存——没记住就再重置一次
3. 如果提示 "Failed to reset root password"，先去 **Main controls → Stop** 把机器关机，再重试

## 第四步：连接 SSH，正式开始使用

有了 IP、端口、root 密码，就可以连接服务器了。

### Windows 用户

推荐使用 **Xshell** 或者 **PuTTY**：


主机（Host）：你的公网 IP
端口（Port）：KiwiVM 里看到的 SSH Port（例如 29876）
用户名：root
密码：刚才重置的 root 密码


### macOS / Linux 用户

打开终端，直接输入：

bash
ssh root@你的IP地址 -p 端口号


例如：
bash
ssh root@123.456.78.90 -p 29876


输入密码后，看到命令行提示符就表示连接成功了。

## 新手最常踩的坑，提前告诉你

**坑1：SSH 连不上**

先进 KiwiVM → Blacklist check，看看是 IP 被封（国内全红国外全绿）还是 VPS 根本没开机。两种情况，两种解法，别一上来就重装系统。

**坑2：流量用超了**

搬瓦工流量是按月计的，用超会暂停服务（显示 "Data transfer limit exceeded"）。每月有个固定重置日，等重置日就行；也可以花小钱提前手动重置。

**坑3：密码忘了**

回 KiwiVM → Root password modification 重新生成一个，下次记好。

**坑4：SSH 端口弄错了**

搬瓦工不是默认 22 端口，是 KiwiVM 首页显示的那个五位数，复制粘贴别手打。
